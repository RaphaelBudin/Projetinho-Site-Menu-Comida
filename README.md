# Projetinho-Site-Menu-Comida
 Projetinho para Treinar conceitos aprendidos em react, assim como arquitetura planejada


 ### Elementos do Projeto ###

- Navbar
    - Nome Loja
    - Botão Carrinho
        - Ícone carrinho
        - Texto carrinho
        - Contador Items

- Plano de Fundo
    - Imagem ocupando os primeiros 2/5  da tela
    - Cor de fundo: cinza chumbo

- Card Cinza
    - Tìtulo
    - Descrição

- Card Branco
    - Produto
        - Nome Item
        - Breve descrição
        - Preço
    - Adicionar Produto
        - Texto
        - Quantia
        - Botão adicionar quantia

- PopUp Carrinho Modal
    - Card Branco
    - Produto
        - Nome Item
        - Preço
        - Quantidade (editável)
    - Alterar quantidade
        - Botão de +
        - Botão de -
    - Total
        - Texto
        - Valor
    - Botões Navegação
        - Fechar
        - Finalizar Pedido
